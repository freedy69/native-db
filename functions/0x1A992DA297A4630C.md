# MISC::ADD_STUNT_JUMP

## Summary
Adds a stunt jump area in the map

## Parameters
* **FLOAT** startMinX
* **FLOAT** startMinY
* **FLOAT** startMinZ
* **FLOAT** startMaxX
* **FLOAT** startMaxY
* **FLOAT** startMaxZ
* **FLOAT** endMinX
* **FLOAT** endMinY
* **FLOAT** endMinZ
* **FLOAT** endMaxX
* **FLOAT** endMaxY
* **FLOAT** endMaxZ
* **FLOAT** cameraPosX
* **FLOAT** cameraPosY
* **FLOAT** cameraPosZ
* **INT** reward: Is the amount of points the player receives if he lands in the end box
* **INT** set = 0:
Is the set to add the stunt jump to.
(eg 0 = singleplayer stuntjumps, 1 = multiplayer stuntjumps) By default only set 0 stunt jumps are enabled.
This value can only be in the range 0 to 31
* **BOOL** camOptional = false

## Returns
* **STUNTJUMP_ID**
