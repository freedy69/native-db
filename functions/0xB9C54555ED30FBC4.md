# ENTITY::PLAY_SYNCHRONIZED_MAP_ENTITY_ANIM

## Summary
Play a synchronized anim on the map entity using the synchronized scene provided

## Parameters
* **FLOAT** posX
* **FLOAT** posY
* **FLOAT** posZ
* **FLOAT** radius
* **MODEL_NAMES** modelHash
* **INT** sceneId: The scene to attach the entity to.
* **STRING** animName: The name of the paired anim to play.
* **STRING** animDictName:
The name of the anim dictionary the anim is in BlendInDelta - the rate at which the task will blend in The blend in duration is 1.0 / BlendInDelta e.g.
1.0 / NORMAL_BLEND_IN -> 1.0 / 8.0 = 0.125 seconds
* **FLOAT** blendDelta
* **FLOAT** blendOutDelta = -8:
The rate at which the task will blend out The blend out duration is -1.0 / BlendOutDelta e.g.
-1.0 / NORMAL_BLEND_OUT -> -1.0 / -8.0 = 0.125 seconds
* **INT** flags = 0
* **FLOAT** moverBlendInDelta = 1000

## Returns
* **BOOL**

## Notes
<!-- AUTOGENERATED: Remove this line before editing -->
Use this to add an entity with a specifically authored anim to a synchronized scene created with CREATE_SYNCHRONIZED_SCENE
Note: as part of this process the entity's collision will be deactivated. To turn it back on again, use SET_ENTITY_COLLISION or pass
a true value to the ActivateCollision parameter of STOP_SYNCHRONISED_ENTITY_ANIM.

p6,p7 probably animname and animdict
