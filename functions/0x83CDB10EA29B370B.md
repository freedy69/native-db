# TASK::TASK_PLAY_ANIM_ADVANCED

## Summary
Plays an anim task on a ped with a reposition and reorientation at the beginning

## Parameters
* **PED_INDEX** ped
* **STRING** animDictName
* **STRING** animName
* **FLOAT** posX
* **FLOAT** posY
* **FLOAT** posZ
* **FLOAT** rotX
* **FLOAT** rotY
* **FLOAT** rotZ
* **FLOAT** blendInDelta = 8:
The rate at which the task will blend in The blend in duration is 1.0 / fBlendInDelta e.g.
1.0 / NORMAL_BLEND_IN -> 1.0 / 8.0 = 0.125 seconds
* **FLOAT** blendOutDelta = -8:
The rate at which the task will blend out The blend out duration is -1.0 / fBlendOutDelta e.g.
-1.0 / NORMAL_BLEND_OUT -> -1.0 / -8.0 = 0.125 seconds
* **INT** timeToPlay = -1
* **ANIMATION_FLAGS** animFlags = 0
* **FLOAT** startPhase = 0
* **EULER_ROT_ORDER** rotOrder = 2
* **IK_CONTROL_FLAGS** ikFlags = 0

## Notes
<!-- AUTOGENERATED: Remove this line before editing -->
As for task_play_anim, but provides the function to specify an inital position and rotation to playback the anim from

It's similar to TASK_PLAY_ANIM, except the first 6 floats let you specify the initial position and rotation of the task. (Ped gets teleported to the position).

Full list of animation dictionaries and anims by DurtyFree: https://github.com/DurtyFree/gta-v-data-dumps/blob/master/animDictsCompact.json
